1.  **Identity Injection**: Assigns specialized expert personas with specific cognitive boundaries.

## üì¶ Getting Started

### Prerequisites

*   **Node.js**: v18 or higher.
*   **API Key**: A valid Google Gemini API Key (get it from [AI Studio](https://aistudio.google.com/)).

### Environment Setup

The application requires an environment variable named `API_KEY`.

#### Mac / Linux:
```bash
export API_KEY="your_api_key_here"
npm run dev
```

#### Windows (PowerShell):
```powershell
$env:API_KEY="your_api_key_here"
npm run dev
```
